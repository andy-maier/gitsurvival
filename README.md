Git survival stuff
==================

Some scripts and guides, without any warranty that they work.

This is really meant for my personal use, but feel free to use it,
or comment on it. Or create issues, if you feel like it.

License: Public domain.

Scripts:

* [`git-archive`](bin/git-archive) - Archive a branch in local repo and its upstream repo.
* [`git-prune`](bin/git-prune) - Delete prunable Git branches in local repo.
* [`git-export`](bin/git-export) - Export a branch as a git bundle. Supports transfer between repos with renames.
* [`git-import`](bin/git-import) - Import a git bundle to prepare for cherry-picking.
